How It Works

This steganography technique uses spaces between words to encode binary data:

  • A single space represents a '0'
  • A double space represents a '1'
Single space (0)
Double space (1)

Example:

Let's encode the letter 'A' (binary: 01000001) into this sentence:

Thequickbrownfoxjumpsoverthelazydog

In this example, the spaces between words encode the binary pattern 01000001, which represents the letter 'A'. The green spaces show where the encoding occurs:

  • Single space after "The" = 0
  • Double space after "quick" = 1
  • Single space after "brown" = 0
  • And so on...
